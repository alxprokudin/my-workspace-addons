# Иконки для проекта "Экспорт в Excel"

Создано **9 вариантов иконок** для проекта в разных стилях:

## 1. `icon-export-excel.svg` - Детальная иконка
**Описание:** Показывает процесс экспорта от Google Sheets (зеленый) к Excel (темно-зеленый) со стрелкой между ними.

**Использование:**
- Размер: 128x128px
- Цвета: Зеленый (Google Sheets), Темно-зеленый (Excel), Синий (стрелка)
- Подходит для: Логотип проекта, иконка в Marketplace

## 2. `icon-simple.svg` - Простая иконка
**Описание:** Минималистичная иконка с документом Excel на градиентном фоне и стрелкой экспорта.

**Использование:**
- Размер: 128x128px
- Цвета: Синий градиент, белый документ
- Подходит для: Favicon, маленькие иконки, карточки проектов

## 3. `icon-minimal.svg` - Минималистичная иконка
**Описание:** Очень простая иконка с таблицей и маленьким бейджем Excel, стрелка экспорта.

**Использование:**
- Размер: 128x128px
- Цвета: Синий градиент, белая таблица
- Подходит для: Иконка в меню, маленькие размеры

## 4. `icon-arrow-down.svg` - Вертикальная композиция
**Описание:** Google Sheets сверху, стрелка вниз, Excel снизу. Зеленый градиентный фон.

**Использование:**
- Размер: 128x128px
- Цвета: Зеленый градиент (Google), белые документы
- Подходит для: Показ процесса экспорта сверху вниз

## 5. `icon-document-flow.svg` - Горизонтальный поток
**Описание:** Два документа рядом (Google Sheets слева, Excel справа) со стрелкой между ними. Светлый фон.

**Использование:**
- Размер: 128x128px
- Цвета: Зеленый (Sheets), темно-зеленый (Excel), светлый фон

## 6. `icon-badge.svg` - С бейджем экспорта
**Описание:** Документ Excel на синем градиентном фоне с бейджем экспорта в углу. Трехцветный градиент.

**Использование:**
- Размер: 128x128px
- Цвета: Синий градиент, белый документ, зеленый бейдж Excel
- Подходит для: Акцент на экспорте

## 7. `icon-modern.svg` - Современный плоский дизайн
**Описание:** Плоский дизайн с тенями, документ с Excel бейджем и плавающей стрелкой экспорта.

**Использование:**
- Размер: 128x128px
- Цвета: Синий градиент, белый документ, тени

## 8. `icon-abstract.svg` - Абстрактная геометрия
**Описание:** Абстрактные геометрические формы, показывающие трансформацию. Минималистичный стиль.

**Использование:**

## Как использовать

### В HTML:
```html
<!-- Как изображение -->
<img src="icon-simple.svg" alt="Export to Excel" width="128" height="128">

<!-- Как фон в CSS -->
.icon {
  background-image: url('icon-simple.svg');
  background-size: contain;
  width: 128px;
  height: 128px;
}
```

### Для Google Workspace Marketplace:
1. Откройте SVG в браузере или редакторе
2. Экспортируйте как PNG 128x128px
3. Используйте в настройках дополнения

### Для favicon:
1. Используйте `icon-simple.svg`
2. Конвертируйте в ICO формат (можно через онлайн-конвертеры)
3. Добавьте в `<head>`:
```html
<link rel="icon" type="image/svg+xml" href="icon-simple.svg">
```
